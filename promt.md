האתר מכיל ארבעה דפים:
1

2

Switch
על כל Card להכיל Switch( לחצן Checkbox מעוצב(.

ה-Switch מסמן את המטבע ככזה שאפשר לצפות בדו"ח זמן אמת לגביו ולקבל המלצת AI לגביו.
כלומר, אם המשתמש בחר בשלושה מטבעות שונים ע"י סימון ה-Switch שלהם, זה אומר שהוא
מעוניין לצפות בדו"חות זמן אמת עבור שלושת המטבעות הללו, או לקבל המלצת AI לגבי שלושת
המטבעות הללו.
אם המשתמש סוגר את הדפדפן וגולש לאתר שוב, על ה- Switch-ים שהוא סימן להמשיך להיות מסומנים.

הצגת המידע מתבצעת ע"י גלישה בכל שנייה ל-API שמחזיר את המידע לגבי כלל המטבעות שסומנו.

חמשת המטבעות אלא יש לבצע בכל שנייה גלישה אחת בלבד המחזירה בבת אחת את המידע של
חמשת המטבעות הדרושים. את המידע יש להחזיר ולהציג רק במטבע דולר )USD)

דף המלצת AI
על דף המלצת AI להציג אך ורק את רשימת המטבעות שהמשתמש סימן ע"י ה-Switch ולאפשר
למשתמש לקבל המלצה לגבי כל אחד מהם.
לדוגמה, ליד כל אחד מהמטבעות הללו להציב לחצן המציג המלצה עבור המטבע שלו.
דוגמה נוספת יכולה להיות הצגת המטבעות הללו ע"י Buttons Radio כך שינתן לבחור רק אחד מהם,
והצבת לחצן אחד ויחיד המציג המלצה לגבי המטבע שנבחר.
ההמלצה צריכה לכלול שני דברים:
א. האם כדאי או לא כדאי לרכוש את המטבע.
ב. פיסקת הסבר המתארת מדוע כדאי או לא כדאי לרכוש את המטבע.
לצורך קבלת ההמלצה יש לבצע גלישה ל-API של ChatGPT, לשלוח Prompt מתאים הכולל מידע
לגבי המטבע, לקבל בחזרה את ההמלצה ולהציגה.
בכדי ש-ChatGPT אכן יוכל לתת המלצה הגיונית, יש לשלוח לו כחלק מה-Prompt את המידע הבא

קישורים ל -APIs
.1 קישור API להחזרת רשימת המטבעות להצגה בדף הבית:
https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd
.2 קישור API להחזרת מידע לגבי מטבע אחד עבור ה-Info More:
https://api.coingecko.com/api/v3/coins/<coin-id>
 לדוגמה, עבור החזרת מידע לגבי Bitcoin יש לגלוש לכתובת:
https://api.coingecko.com/api/v3/coins/bitcoin
.3 קישור API להחזרת שערי מטבעות עבור הצגה בגרף דו"חות זמן אמת:
https://min-api.cryptocompare.com/data/pricemulti?tsyms=usd&fsyms=<symbols>
 לדוגמה, עבור החזרת מידע לגבי המטבעות Bitcoin, Ethereum ו-Doge יש לגלוש לכתובת:
https://min-api.cryptocompare.com/data/pricemulti?tsyms=usd&fsyms=btc,eth,doge
.4 קישור API להחזרת מידע לגבי מטבע אחד עבור המלצת ה-AI:
https://api.coingecko.com/api/v3/coins/<coin-id>?market_data=true
 לדוגמה, עבור החזרת מידע לגבי Bitcoin יש לגלוש לכתובת:
https://api.coingecko.com/api/v3/coins/bitcoin?market_data=true
.5 קישור API ל-ChatGPT:
https://api.openai.com/v1/chat/completions
